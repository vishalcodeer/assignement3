GAME NAME- GUESS THE NO.


 ABOUT THE GAME
Guess the secret number (1–100). The computer picks a number. You try to find it. Every try counts. Your best score is saved forever. Beat your old self!

Features
• Clean & simple C code
• Random number every game
• Tells you “Too High” / “Too Low”
• Saves best attempts in a file
• Shows current record
• Beginner-friendly

 How to Play

Game chooses a random number from 1 to 100.

You type guesses one by one.

Messages guide you:

Too low

Too high

Correct!


 CONTROLS
Just type a number → press Enter.
That's it.


 Winning & Losing
✔ Win
You guess the number → NEW RECORD (if attempts are best).
